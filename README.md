## Installation

You can install the package via composer:

```bash
composer require shah-07/greetings-generator
```

## Usage

```php
$skeleton = new Shah-07\GreetingsGenerator();
echo $skeleton->echoPhrase('Hello, Shah-07!');
```

## Testing

